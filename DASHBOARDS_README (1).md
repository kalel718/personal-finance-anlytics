# Dashboard Screenshots

## Instructions

Add your Tableau dashboard screenshots here to showcase your visualizations in the README.

### Recommended Screenshots:

1. **Full Dashboard View**
   - Filename: `dashboard_full_view.png`
   - Description: Complete view of all visualizations together

2. **Spending by Category**
   - Filename: `spending_by_category.png`
   - Description: Pie chart showing expense distribution

3. **Income vs Expenses Trend**
   - Filename: `income_vs_expenses.png`
   - Description: Line chart showing monthly cash flow

4. **Top Merchants**
   - Filename: `top_merchants.png`
   - Description: Bar chart of highest spending locations

### How to Capture Screenshots:

**From Tableau Public:**
1. Open your published dashboard
2. Use screenshot tool (Windows: Snipping Tool, Mac: Cmd+Shift+4)
3. Save as PNG format
4. Upload to this folder

**Alternative:**
1. In Tableau Desktop: Dashboard > Export Image
2. Choose PNG format
3. Save to this folder

### Tips for Quality Screenshots:
- Use high resolution (at least 1920x1080)
- Ensure all labels are readable
- Capture the entire visualization without cutoffs
- Use consistent naming convention
